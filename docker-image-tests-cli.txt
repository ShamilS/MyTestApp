docker inspect myapp | grep -i port

=== How to login to GHCR ---

Step 1: Login to GHCR

First generate a PAT token on GitHub:

GitHub → Settings → Developer Settings 
    → Personal Access Tokens → Tokens (classic) 
    → New token →  check read:packages → Generate

Then login:

bash> echo YOUR_PAT_TOKEN | docker login ghcr.io -u {GitHub user name} --password-stdin

You should see: Login Succeeded
